# ucms project

## Project Overview

This project is a React application built with Vite, TypeScript, Tailwind CSS, and shadcn-ui.  
It is a cleaned version of the previous project, free from any external or unwanted dependencies.

---

## Features

- Fast development with Vite
- Type-safe code using TypeScript
- Modern UI with Tailwind CSS and shadcn-ui components
- Easy local development and live preview

---

## Technologies Used

- **Vite** – build tool and development server
- **React** – frontend library
- **TypeScript** – static type checking
- **Tailwind CSS** – utility-first CSS framework
- **shadcn-ui** – prebuilt UI components
